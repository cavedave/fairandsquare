    FAQ

    What does the fair and square process aim to do?

    The Fair and Square task division process allows a group of people to divide a number of tasks between them. The division of tasks process is fair, transparent and quick which reduces the amount of work that people must do.

    How do you decide who should get each task?

    The allocation is done in a way that gives everyone the same amount of perceived work and minimises that amount. Any other way of allocating taks will mean that someone has to work more than everyone does currently

    What do you mean by fair

    With two people each aims expects to have to work at most 50% of what they think all the tasks involve. This 50% is a fair share. Any less than this they get is an improvement the tool has found that reduces peoples perceived workload.

    Why do you use points rather than time based values?

    Some things are unpleasant but quick. We measure perceived effort and unpleasantness in one metric: how the task weighs up in comparison to the others.

    Do I have to do what the system says

    No, the suggested allocation can be thought of as an initial start point for decisions. By using an independant algorithm everyone can see their opinion being taken into account and that the process was impartial.

    How many tasks can be entered?

    As many tasks as you want. But for a large number the system will become time consuming. The ten worst tasks are probably the ones people care about most. You probably know what taks people hate so treat them as the main issues which need to be addressed first. After these are equitably divided a simple pick a task in turn rule should be seen to be fair.

    What if one task is really valuable?

    Sometimes one task is almost all the work. Using our process we can find this out quickly. This one task can then be divided out amongst many people. This removes the problem of one person having to do the big unpleasant task and others not appreciating exactly how much work this involves.

    What if a task cannot be divided out?

    Maybe a task takes so much preparation that it is only practical for one person to do it. If you have this problem email us at info@fairandsquare.ie. We are trialling a system to help with this more complicated problem.

    Thank you for using the Fair and Square task division process. Please feel free to contact us with any feedback or suggestions
